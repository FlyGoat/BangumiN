# BangumiN
BangumiN - 使用Angular，Node.js和NoSQL构建的PWA Bangumi客户端。

![name](./images/name.png)

# 注意☠

这个项目仍处于初期开发阶段，功能随时可能在没有通知的情况下发生变化。

如果您发现了任何错误，安全漏洞或任何其他异常，欢迎告知！


# 安装


* 前端
    * `npm install`
    * `npm run start-dev`
* 后端
    * `npm install`
    * `npm run start-dev`


# 功能
* 作品的搜索，展示，进度管理，状态修改
* 用户/作品评分数据可视化
* 剧透箱：安全的发表剧透
